UpdateRetrievalMode

The UpdateRetrievalMode enumeration contains retrieval modes for updates in a local repository.

Because update extended attributes are large, they don't get loaded when a repository is opened. They can be retrieved by calling one of the GetUpdates method for a IRepository, like GetUpdate(Identity, UpdateRetrievalMode) or GetUpdates(UpdateRetrievalMode)
